# Financial Message Authentication System

**Компания:** Innotech Group  
**Позиция:** Principal Analyst  
**Период:** Nov 2023 – Jun 2025

## Цель проекта
Обеспечение соответствия требованиям Центрального банка РФ (SPFS) для безопасной аутентификации финансовых сообщений.

## Основные задачи
- Интеграция 2 систем
- Документирование требований и архитектуры
- Тестирование и развертывание в продуктивной среде

## Результаты
- Полное соответствие требованиям SPFS
- Стабильная работа системы в продакшене

## Технологии
Kubernetes, Kafka, PostgreSQL, REST APIs
