# Long-Term Storage System for Financial Messages

**Компания:** Innotech Group  
**Позиция:** Principal Analyst  
**Период:** Nov 2023 – Jun 2025

## Цель проекта
Миграция базы данных Oracle (40 TB) в ClickHouse для долгосрочного хранения финансовых сообщений и интеграции с BI.

## Основные задачи
- Архитектура хранения и потоков данных
- Проектирование отчетности и BI-интеграция
- AS-IS / TO-BE анализ и документация

## Результаты
- Успешная миграция 40 TB данных
- Интеграция с BI для построения отчетности
- Повышение производительности системы хранения

## Технологии
ClickHouse, PostgreSQL, Kafka, REST APIs, BI tools
