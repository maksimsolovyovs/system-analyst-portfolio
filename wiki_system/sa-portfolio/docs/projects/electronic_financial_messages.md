# Electronic Financial Message Processing & Routing System (SWIFT)

**Компания:** Innotech Group  
**Позиция:** Principal Analyst  
**Период:** Nov 2023 – Jun 2025

## Цель проекта
Миграция устаревшей монолитной системы обмена финансовыми сообщениями в микросервисную архитектуру с унифицированной доставкой сообщений через message broker.

## Основные задачи
- Интеграция 15 существующих систем
- Архитектура микросервисов
- Миграция IBM WebSphere MQ → Apache ActiveMQ
- AS-IS / TO-BE анализ и документация
- Архитектурные модели (C4, UML, BPMN)

## Результаты
- Успешная интеграция всех систем
- Упрощение маршрутизации сообщений
- Повышение отказоустойчивости системы
- Улучшение мониторинга через Prometheus, Grafana, Kibana

## Технологии
Kubernetes, Kafka, Elasticsearch, PostgreSQL, IBM WebSphere MQ, Apache ActiveMQ, REST APIs, Prometheus, Grafana, Kibana
